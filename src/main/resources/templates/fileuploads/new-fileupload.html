CSV Format Guidelines
The following five lines in a CSV file consist of:
1) A header record (identified by the value 0 in column 1)
2) Three measurement records (identified by the value 1 in column 1)
3) A trailer record (identified by the value 999 in column 1)
0,ANT007,B,N,VSWR
1,145.700,1.52
1,145.725,1.60
1,145.750,1.65
999,5
Notes:
1) Only one header record per file which always has five columns:
- Record identifier in column 1
- Antenna code in column 2
- Antenna type in column 3
- Base antenna code or 'N' if none in column 4
- Measurement code in column 5
2) Multiple measurement records which always have three columns:
- Record identifier in column 1
- Measuring (determinant) value in column 2
- Measured (determined) value in column 3
3) Only one trailer record per file which always has two columns:
- Record identifier in column 1
- Number of records in the CSV file, including header and trailer
